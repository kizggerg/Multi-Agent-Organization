# Cost and Budgeting

## Instructions

1. Estimate costs in three buckets: delivery, runtime, agent usage.
2. State assumptions, range, and confidence level.
3. Compare current estimate to prior baseline and show variance.
4. Flag top cost drivers and cost-reduction options.
5. Highlight when VP approval is needed.

## Output Template

- Cost summary (delivery/runtime/agent)
- Baseline vs current variance
- Confidence level and assumptions
- Cost driver analysis
- Recommendation and escalation status
