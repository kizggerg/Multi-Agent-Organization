# DevOps Release

## Instructions

1. Verify build, test, and deployment pipeline health.
2. Define rollout strategy (phased/canary/full) and rollback plan.
3. Confirm observability and alerting for new behavior.
4. Validate release checklist completion before go-live.
5. Record release outcomes and follow-up actions.

## Output Template

- Release plan
- Rollback strategy
- Readiness checklist status
- Monitoring/alert checks
- Post-release notes
